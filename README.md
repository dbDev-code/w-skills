# Skills 技能库总览

当前共有 **20 个技能**，按功能领域可分为以下几大类：

---

## 一、LangGraph/LangChain 生态（7个）

| 技能名称 | 功能描述 |
|---------|---------|
| **langgraph-project-setup** | LangGraph 项目初始化，生成项目结构、`langgraph.json` 配置、环境变量模板 |
| **langgraph-state-management** | 状态模式设计、Reducer 实现、持久化配置（InMemory/Sqlite/Postgres） |
| **langgraph-agent-patterns** | 多智能体协调模式：Supervisor、Router、Orchestrator-Worker、Handoffs |
| **langgraph-error-handling** | 错误处理策略：RetryPolicy、LLM 恢复循环、Human-in-the-loop 升级 |
| **langgraph-testing-evaluation** | 测试与评估：单元测试、轨迹评估、LangSmith 数据集评估、A/B 对比 |
| **langsmith-deployment** | LangSmith 部署：Cloud/Hybrid/Standalone 三种模式、CI/CD 集成 |
| **langsmith-trace-analyzer** | 追踪分析：下载、组织、分析 LangSmith traces 用于调试和评估 |

---

## 二、Deep Agents 智能体框架（2个）

| 技能名称 | 功能描述 |
|---------|---------|
| **deepagents-setup-configuration** | Deep Agents 项目初始化（Python/JS），内置规划、文件系统、子智能体等中间件 |
| **deepagents-planning-todos** | 使用 `write_todos` 进行任务规划和分解，适合多步骤复杂任务 |

---

## 三、前端开发（7个）

| 技能名称 | 功能描述 |
|---------|---------|
| **frontend-design** | 创建独特、高质量的前端界面，避免通用 AI 美学，注重创意和设计细节 |
| **react-best-practices** | React/Next.js 性能优化指南，40+ 规则涵盖瀑布消除、Bundle 优化、渲染性能等 |
| **ui-ux-pro-max** | UI/UX 设计智能库：50 种样式、21 种调色板、字体配对、图表类型、多框架支持 |
| **cache-components** | Next.js Cache Components 和 Partial Prerendering (PPR) 专家指导 |
| **web-design-guidelines** | Web Interface Guidelines 合规审查，检查 UI 代码的可访问性、最佳实践、UX 设计规范 |
| **react-native-skills** | React Native/Expo 性能优化指南，35+ 规则涵盖列表性能、动画、导航、原生模块等 |
| **composition-patterns** | React 组合模式，避免布尔属性泛滥，使用复合组件、状态提升、依赖注入构建可扩展组件 |

---

## 四、AI 内容生成（2个）

| 技能名称 | 功能描述 |
|---------|---------|
| **seedance** | Seedance 2.0 视频提示词生成器，为字节跳动即梦平台生成高质量中文视频提示词，支持 15 秒视频生成、视频延长、一镜到底等功能 |
| **nano-banana-pro** | Google Gemini 3 Pro Image 图像生成/编辑，支持文生图、图生图，1K/2K/4K 分辨率，提供 Draft→Iterate→Final 工作流 |

---

## 五、元技能（2个）

| 技能名称 | 功能描述 |
|---------|---------|
| **find-skills** | 技能发现助手，帮助用户搜索和安装开放技能生态中的技能，支持 `npx skills find/add` 命令 |
| **skill-creator** | 创建新技能的指南，包含技能结构、设计原则、创建流程、打包发布等完整指导 |

---

## 技能库特点

1. **LangGraph 全流程覆盖**：从项目初始化 → 状态管理 → 多智能体模式 → 错误处理 → 测试评估 → 部署上线 → 追踪分析

2. **Deep Agents 简化开发**：提供更高层的抽象，内置规划、文件系统、子智能体等功能

3. **前端设计专业化**：涵盖性能优化、设计美学、UI/UX 最佳实践

4. **AI 内容生成**：支持 Seedance 2.0 视频生成 + Gemini 图像生成/编辑完整工作流

5. **可扩展性**：通过 `find-skills` 发现新技能，通过 `skill-creator` 创建自定义技能
