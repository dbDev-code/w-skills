## 技能库特点

1. **Deep Agents 简化开发**：提供更高层的抽象，内置规划、文件系统、子智能体等功能

2. **规划与设计先行**：通过 brainstorming 头脑风暴和 write-a-prd 创建 PRD，确保设计方案在实现前获得认可

3. **前端设计专业化**：涵盖性能优化、设计美学、UI/UX 最佳实践

4. **AI 内容生成**：支持 Seedance 2.0 视频生成 + Gemini 图像生成/编辑完整工作流

5. **开发工具完善**：提供代码审查、PR 创建、文档更新、Web 应用测试等完整开发流程辅助工具

6. **可扩展性**：通过 `find-skills` 发现新技能，通过 `skill-creator` 创建自定义技能

7. **全栈开发覆盖**：提供前端、后端、数据库、部署等全栈开发领域的专业知识
