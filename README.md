# Skills 技能库总览

当前共有 **30 个技能**，按功能领域可分为以下几大类：

---

## 一、规划与设计（2个）

| 技能名称 | 功能描述 |
|---------|---------|
| **brainstorming** | 创意工作前的头脑风暴工具，通过协作对话将想法转化为完整的设计和规格，在实现功能前必须先获得设计批准 |
| **write-a-prd** | PRD 创建工具，通过用户访谈、代码库探索和模块设计创建产品需求文档，并作为 GitHub issue 提交 |

## 技能库特点

1. **规划与设计先行**：通过 brainstorming 头脑风暴和 write-a-prd 创建 PRD，确保设计方案在实现前获得认可

2. **前端设计专业化**：涵盖性能优化、设计美学、UI/UX 最佳实践

3. **AI 内容生成**：支持 Seedance 2.0 视频生成 + Gemini 图像生成/编辑完整工作流

4. **开发工具完善**：提供 Karpathy 编码准则、代码审查、PR 创建、文档更新、Web 应用测试等完整开发流程辅助工具

5. **可扩展性**：通过 `find-skills` 发现新技能，通过 `skill-creator` 创建自定义技能

6. **全栈开发覆盖**：提供前端、后端、数据库、部署等全栈开发领域的专业知识
